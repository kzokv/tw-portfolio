State 5 · Server-side write guard (defense-in-depth)

Client-side, CTAs are hidden when isSharedContext === true. If a request reaches the server with sessionUserId !== contextUserId on a portfolio-write endpoint (e.g. direct API call, race with stale UI), server rejects.

POST /transactions
Cookie: tw_context_user_id=alice-uid; session=keith.v1.sig
HTTP/1.1 403 Forbidden
Content-Type: application/json
{
"error": "write_blocked_viewing_shared",
"message": "Writes are disabled while viewing a shared portfolio."
}
