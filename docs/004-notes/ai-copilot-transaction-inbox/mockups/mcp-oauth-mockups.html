Connector flow

1
Enable Developer Mode

Settings -> Apps -> Advanced settings.

2
Add custom app

Paste the Vakwen public MCP endpoint.

3
Start OAuth

ChatGPT uses OAuth discovery and PKCE.

4
Approve in Vakwen

Choose lifetime and scopes on the consent page.

5
Use tools

Return to ChatGPT after tool scan completes.
