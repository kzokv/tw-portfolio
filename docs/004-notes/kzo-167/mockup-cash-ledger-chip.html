Visual contract

  • Account name — primary text weight, no decoration. Replaces the raw ${accountId}.
  • TWD — currency code rendered in rounded-full bg-slate-100 font-mono text-[11px]. Untranslated by industry convention; mono face signals code, not prose.
  • Broker — type label rendered in rounded-full bg-blue-50 text-blue-700. Localized via accountTypeBroker / accountTypeBank / accountTypeWallet in cash-ledger/i18n.ts.

What changes vs current

  • Filter dropdown — was {accountId} (raw); becomes {name} (TWD · Broker).
  • Summary chip header — was {accountId} / {currency}; becomes {name} + currency-mono chip + type pill.
  • Table account column — upgraded to {name} only (rows already crowded; no chip there). Implementer judgment, scope-todo confirms this is OK.
