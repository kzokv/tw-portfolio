# Taiwan Broker Public Commission Rates

Use this file when the task requires a practical reference for publicly visible Taiwan brokerage commission discounts.

Last refreshed: March 4, 2026

## Important Limits

- Taiwan brokers do not all charge the same stock commission.
- TWSE allows brokers to set customer fee schedules, discounts, and minimum per-order fees based on their own standards and customer arrangements.
- There is no official TWSE or FSC page that publishes one complete live table of every broker's actual retail discount.
- Many brokers publicly show only the standard board rate and handle customer discounts through branch agreements, campaign pages, or account-opening offers.
- The table below is a market reference compiled from a public third-party comparison page, not an official regulator-maintained directory. Use it as a starting point, not as binding evidence of the user's contracted rate.

## Official Rule Baseline

- TWSE 2008 notice: brokers may set fee standards by customer transaction amount and may also set discounts and minimum fees.
  - https://www.twse.com.tw/staticFiles/marketAnnounce/setAnnounce/0970003165.htm
- TWSE 2016 notice: brokers report e-trading and non-e-trading commission income monthly, reinforcing that channel-specific pricing can differ.
  - https://www.twse.com.tw/docs1/data01/set/public_html/1050502041.htm
- Common public board rate reference: `1.425‰` before broker-specific discount.

## How To Use This Table In Answers

- If the user asks "all brokers", state clearly that this is the broadest publicly compiled comparison found, but it is not an official exchange-maintained master list.
- If the user needs a binding answer for one broker, prefer that broker's official fee page, campaign page, or account agreement over this table.
- If the user asks for bookkeeping assumptions, store:
  - board rate assumption
  - discount assumption
  - minimum fee assumption
  - whether the fee is charged upfront, rebated later, or negotiated off-table

## Suggested Output Wording

Use wording like:

`券商手續費可能依客戶等級、電子或人工下單、活動專案與個別議價而不同；若無對帳單或合約，僅能先以公開牌告或活動價作為記帳假設。`
